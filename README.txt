Compilers used: g++ (Windows/Linux) and clang++ (Mac)
Both compile the same file with no issues.

To run: simply compile the tictactoe.cpp file and run.

1.) Enter 1 to play first or 2 to play second
2.) Enter a number between 1 & 3 to select a row to play in
3.) Enter a number between 1 & 3 to select a column to play in
4.) Repeat steps 2 & 3 until the game finishes
5.) Enter y to play again or n to exit the program
